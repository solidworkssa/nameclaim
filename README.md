# NameClaim

Decentralized name registration and resolution.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/NameClaim.sol`
- Stacks: `contracts/stacks/contracts/nameclaim.clar`

## Setup
```bash
pnpm install
pnpm dev
```
